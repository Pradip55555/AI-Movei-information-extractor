# 🎬 CineMind AI – Movie Information Extractor

CineMind AI is an AI-powered Movie Information Extractor built using Streamlit, LangChain, and Mistral AI.
The application converts movie descriptions into structured JSON data using Large Language Models (LLMs).

---

# 🚀 Features

✅ AI-powered movie analysis
✅ Extracts structured movie data
✅ Beautiful futuristic UI
✅ JSON export functionality
✅ Smart genre and cast detection
✅ Interactive dashboard design
✅ Download extracted data as JSON
✅ Real-time AI processing animations

---

# 🛠 Technologies Used

* Python
* Streamlit
* LangChain
* Mistral AI
* Pydantic
* HTML/CSS
* dotenv

# 📦 Extracted Information

The AI extracts:

* 🎬 Movie Title
* 📅 Release Year
* 🎭 Genre
* 🎥 Director
* 👨‍🎤 Cast
* ⭐ IMDb Rating
* 📝 Summary

## 3️⃣ Create Virtual Environment

# 🧠 AI Workflow

1. User enters movie description
2. LangChain creates structured prompt
3. Mistral AI processes the text
4. Pydantic validates JSON schema
5. Streamlit displays structured output
